# ux-e2e

端到端网页体验测评。基于用户提供的 2-4 个关键页面（产品详情页、购买页、控制台等），生成覆盖完整客户旅程的专业测评报告。

## 核心框架

### 客户旅程四阶段

感知（Awareness）→ 购买（Purchase）→ 使用（Usage）→ 反馈（Feedback）

每个阶段有专属的评估维度和 SMB 客户特征检查项。

### 五层评价维度

- **视觉层** — 布局、色彩、排版、图标一致性
- **交互层** — 费茨定律、希克定律、格式塔原则、系统反馈
- **文案层** — 价值主张、CTA 清晰度、术语解释、价格透明
- **信息架构层** — 内容组织、导航结构、搜索效率
- **战略层** — 用户目标匹配、商业平衡、SMB 适配度

## 输出

- 综合评分仪表盘（六维度星级评分）
- 分页面详细评估（每页五层维度逐项打分）
- 交互设计原则专项诊断
- 转化漏斗分析（各阶段流失点）
- P0/P1/P2 问题清单 + 改进路线图
- 竞品对比参考（如有）
- 响应式 HTML 报告

## 适用场景

- 新产品上线前全链路体验体检
- 购买转化率优化分析
- 控制台操作效率评估
- 竞品体验差距分析
