# ux-comparative-workflow

串行化竞品对比体验测评工作流。四步引导式流程，结合 ux-evaluator 生成含竞品对比矩阵的完整 UX 测评报告。

## 功能

1. **Step 1** — 接收待测评的产品/页面 URL
2. **Step 2** — 收集竞品信息（URL 自动抓取或用户描述）
3. **Step 3** — 用户补充背景信息（可选）
4. **Step 4** — 输出含左侧导航栏的专业 HTML 测评报告，包含竞品对比速览表

## 输出

- 四维度评分（视觉/交互/文案/转化漏斗）
- 21 子维度详细分析，每条附 UX 原则引用
- P0/P1/P2 优先级改进建议
- 竞品对比矩阵（横向对比三家）
- 响应式 HTML，移动端友好

## 前置条件

- 依赖 `ux-evaluator` skill 的评分体系和报告模板
